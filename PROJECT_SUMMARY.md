## 🔌 API Endpoints

### POST /api/contact
**Description**: Submit contact message
```bash
curl -X POST http://localhost:3000/api/contact \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "message": "Hello, I loved your portfolio!"
  }'
```

**Response**:
```json
{
  "success": true,
  "message": "Message received successfully",
  "data": {
    "id": "1234567890",
    "name": "John Doe",
    "email": "john@example.com",
    "message": "Hello, I loved your portfolio!",
    "timestamp": "2024-04-17T10:30:00Z"
  }
}
```

### GET /api/contact (Optional)
**Description**: Retrieve all messages (for admin purposes)

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| Port in use | Use different port: `npm run dev -- -p 3001` |
| Module not found | Reinstall: `rm -rf node_modules && npm install` |
| Dark mode not working | Check localStorage enabled in browser |
| Contact form errors | Check API route accessible at `/api/contact` |
| Slow build | Check for large dependencies, optimize images |
